Facilities Management Capstone

Origin:

Final Capstone Project consisting of 4 senior team members and a graduate student at University of Arkansas at Little Rock.

Purpose:

To move 20 years worth of billing records at Facilities Management from an excel to a relational database. Also provide forms for easier entry for their accountant.

Project Length:

1 Year

Languages/Software Used:

Javascript, HTML, CSS, ASP.NET, SQL

View:

We created 3 similar forms for entering gas, water, and electric. We made sure forms were either set to take date, numbers, or drop downs to prevent SQL Injection. Even though only one person is using the system, their accountant, we made sure that you can't type char characters. Below Account ID is a drop down which pulls currently available accounts from the SQL Server Database. Same goes for Meter ID.

Below each form were tables that showed what was previously entered

When the accountant clicks submit, the data is entered to a pre_billing table which has a stored procedure batch procedure that runs every night to insert data to the main database. The accountant can insert data for the account multiple times if he/she messes up, but the stored procedure will only insert the latest/current one for that day.

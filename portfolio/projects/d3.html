Visualization of Texas Death Row Executions

Origin:

Data Visualization Graduate Course Project where we had to select a dataset and use or build visualization software to display interesting things about the data. Selected dataset was Texas Executions from Kaggle.com

Purpose:

Visualize:

-main counties of convictions for Texas Counties

-most frequent words in a executioned person's last statement

-crime history and gender of executioned people

Project Length:

3 Months

Languages/Software Used:

Javascript, D3.js Library

View:

Texas County Execution Conviction Counties<--wait 15 seconds for map to load

Last Statements Frequent Words

Gender/Previous Crimes Pie Graph

The d3.js libary mainly involed drawing the svg object, opening data files, looping through the file, extracting the data, and putting into the svg to draw a county map, or word cloud. The main problem with d3.js is that it uses a lot of ram drawing svg's, so working with big data is not advisable. I suggest aggregating the data first or your browser will crash.
